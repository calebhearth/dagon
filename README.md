## Cut to the chase

```
fibonacci-recursive(n): # fuckin hyphens instead of underscores.
  if n <= 1, return n  #leading conditionals. comma rather than semicolon to separate statements
  self(n - 1) + self(n - 2)
```

```
fibonacci-imperative(n):
  current: 0
  next: 1

  n.times
    temp: next
    next +: current
    current: temp

  current
```

## Classes

/^(?:[A-Z][a-z]+)+:$/

```
MyClass:
  # here lies the definition

Tree(Plant):

```

## Methods

/^(?:[a-z]+-?)+[^-][\(\)]?:\n  $/

```
do-stuff:
  # definition
stuff:
  # definition
print():
  # definition
```
